# 狼人殺小助手

- 支持阿瓦隆自动语音，无需法官
- 支持狼人杀创建房间（板子预女猎白，守卫）
- 狼人杀查看每晚信息
- 狼人杀无需法官，自动语音

## Environment

`Node >= 6`

## Start

 - Clone or download this repository
 - Enter your local directory, and install dependencies:

``` bash
yarn
```

## Develop

``` bash
npm run dev
```

## Build

``` bash
npm run build
```
